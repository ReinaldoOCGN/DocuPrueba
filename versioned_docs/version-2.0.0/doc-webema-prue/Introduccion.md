# Introducción

Este proyecto consiste en una aplicación web desarrollada en ASP.NET Core MVC, cuya finalidad es conectarse a una base de datos SQL Server para consultar y mostrar información almacenada en la tabla EmaUsuario.

La aplicación está estructurada siguiendo el patrón Modelo–Vista–Controlador (MVC), permitiendo una separación clara entre:

La lógica de acceso a datos.

La lógica de negocio.

La presentación de la información.

Actualmente, el sistema está diseñado únicamente para la consulta y visualización de datos, aunque su estructura permite que en el futuro se puedan agregar funcionalidades como inserción, actualización o eliminación de registros.
